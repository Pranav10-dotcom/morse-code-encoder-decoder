# 🔠 Morse Code Translator

A simple Python-based Morse Code Translator that converts:

- ✅ Text → Morse Code
- ✅ Morse Code → Text

## Features

- Translate English text to Morse code
- Decode Morse code back to English
- Supports:
  - A-Z
  - Numbers (0-9)
  - Spaces between words

## Technologies Used

- Python 3

## How to Run

Clone the repository

```bash
git clone https://github.com/YourUsername/morse-code-translator.git
```

Go inside the folder

```bash
cd morse-code-translator
```

Run

```bash
python morse_translator.py
```

## Example

```
MORSE CODE TRANSLATOR

1. Text to Morse
2. Morse to Text

Enter choice: 1

Enter text:
HELLO

Output:
.... . .-.. .-.. ---
```
